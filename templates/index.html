Welcome to the MultiMC instances repository!

Share your exported multimc instances with your friends here

Using this website

This website is designed to make sharing minecraft instances exported using multimc easier. You can upload an exported instance here and give the link to your friends. MultiMC supports importing an exported instance via a direct download url that this server will provide.

Uploading

To upload an instance, navigate to the upload page, give your instance a name, click "Choose File" to select the exported instance file from MultiMC, then click "Upload Instance" to submit the form. You will be taken to the unique page generated for your instance where you can copy the download link to share with friends.

Viewing instances

You can view all of the uploaded instance by navigating to the view page.

Download instances

From the view page and the instance detail page provide links to download the exported MultiMC instance. You can copy this link and paste it into MultiMC as the url for a direct download when importing as a zip.
